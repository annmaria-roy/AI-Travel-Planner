# ✈️ AI Travel Planner

An AI-powered travel planning web application built using **FastAPI**, **Groq AI**, **HTML**, **CSS**, and **JavaScript**. The application generates personalized travel itineraries based on the user's destination and travel preferences.

## 🌐 Live Demo

https://ai-travel-planner-e998.onrender.com

## 📂 GitHub Repository

https://github.com/annmaria-roy/AI-Travel-Planner

- 🤖 AI-generated travel plans

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/annmaria-roy/AI-Travel-Planner.git
```

Go to the project folder

```bash
cd AI-Travel-Planner
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

Run the application

```bash
uvicorn main:app --reload
```

Open your browser

```
http://127.0.0.1:8000
```
